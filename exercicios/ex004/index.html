Lista de Definições

HTML

Linguaguem de marcação para a criação do conteúdo de um site. HTML Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS

Linguagem de marcação para a criação do design de um site. O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML).

JavaScript

Linguagem de programação para a criação de interatividade de um site. JavaScript é uma linguagem de programação orientada a objetos, multiplataforma. Foi criada em 1995 por Brendan Eich enquanto trabalhava na Netscape Communications Corporation. Originalmente projetada para rodar no Netscape Navigator, ela tinha o propósito de tornar determinados processos de páginas web mais dinâmicos. JavaScript é uma linguagem de programação de alto nível utilizada por bilhões de páginas para adicionar funcionalidades, verificar formulários, comunicar com servidores, entre outros.
